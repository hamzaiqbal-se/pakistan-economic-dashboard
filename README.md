# 🇵🇰 Pakistan Economic Indicators Dashboard

An interactive data dashboard built with Python and Streamlit that analyzes Pakistan's key economic trends from 2000 to 2025, with a Machine Learning model to predict future inflation.

## 📊 Features

- Interactive line charts for any economic indicator
- GDP Growth vs Inflation comparison
- PKR/USD exchange rate depreciation trend
- Exports vs Imports analysis
- Correlation heatmap between all indicators
- **Machine Learning inflation predictor** (Linear Regression)
- Year range filter and indicator selector

## 🛠️ Tech Stack

- Python, Streamlit, Pandas, Plotly, Scikit-learn

## 📁 Dataset

- Source: Kaggle — Pakistan Economic Indicators 2000-2025
- 26 years of data across 32 economic indicators

## 🤖 Machine Learning

- Model: Linear Regression
- Predicts future inflation rate based on historical year-over-year trend
- Includes a transparency note on model limitations

## 🚀 How to Run

​```bash
pip install -r requirements.txt
streamlit run app.py
​```

## 💡 Key Insights

- Highest inflation recorded in 2023 at 29.2%
- Best GDP growth was in 2005 at 9%
- PKR dropped significantly against USD over 25 years
- Imports consistently exceeded exports creating trade deficit
- Model predicts continued inflation trend for 2026-2030
